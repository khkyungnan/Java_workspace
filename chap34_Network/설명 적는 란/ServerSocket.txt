ServerSocket
	소켓 통신을 구현할 때 사용되는 클래스
	서버 측에서 클라이언트의 연결을 수락하고
	데이터를 주고받을 수 있게 해줌
	서버를 구현하면 클라이언트가 서버에 연결할 수 있음
	서버는 클라이언트와 통신할 수 있음
	
	메서드						      /*서버 포트번호*/
	ServerSocket sS = new ServerSocket( int 포트번호 ) 
		지정된 포트 번호로 서버 소켓을 생성
	accept() : 클라이언트의 연결을 기다리고, 클라이언트가 연결되면
				클라이언트와 통신할 수 있는 소켓을반환
	 close() : 서버 소켓을 닫음
